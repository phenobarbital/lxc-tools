LXC-tools

LXC es un sistema de contenedores para virtualizar GNU/Linux en espacios aislados (isolated). Dentro de los contenedores reside otra versi칩n de GNU/Linux, la cual:

  • Posee su propia interfaz de red
  • Se pueden aplicar cuotas de disco/CPU/RAM
  • Se pueden detener, apagar y/o suspender

Es un "chroot" mejorado puesto que las facilidades de administraci칩n y ejecuci칩n basadas en chroot (lxc-execute) pueden ser utilizadas dentro de un contenedor LXC.

LXC-tools es un conjunto de scripts que permiten crear contenedores personalizados de GNU/Linux Debian (y otras distribuciones) dentro de un contenedor LXC.

Emula el comportamiento de los Debian Xen-tools, permitiendo personalizar el contenedor, ejecutar roles, ejecutar scripts dentro de los contenedores, etc.
